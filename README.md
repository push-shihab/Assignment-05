1️⃣ What is the difference between var, let, and const?
===> var is hoisted. means it can be accessed from anywhere in the file. it is accessible in block scope and also out of the block scope. we can redifine, reassign and can initilize without the value.
let is also hoisted but cannot initialize without the value. it is block scoped. eventhough we cannot redefine it but we can redeclare it.
const is block scooped. we cannot redifine it or redeclare it. we must put a value in it like let
